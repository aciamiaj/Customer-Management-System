# Customer-Management-System

This program implements a simple Customer Management System using Java Swing. It allows users to add new customers, update existing customer information, and view a list of all customers.

Prerequisites

- Java Development Kit (JDK) 8 or above
- IDE or compiler to run Java code

sage

1. Clone the repository or download the source code files.
2. Open the project in your Java IDE or compiler.
3. Build and run the `CustomerPage` class.

Features

- **Add Customer**: Enter customer details such as ID, name, phone number, email, and postal code. Click the "Add" button to add the customer to the system.

- **Update Customer**: Enter the customer ID and click the "Find" button to retrieve the customer's details. Modify the customer's information as needed and click the "Update" button to save the changes.

- **View Customers**: Click the "View" button to display a list of all customers in the system.

File Storage

Customer data is stored in a file named `customer.dat`. The file format is as follows:

ID,Name,Phone,Email,Postal Code


Each customer record is stored on a new line.
